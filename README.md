# AI Sports Talent Scout

> An AI-powered Machine Learning application that analyzes cricket player performance data to identify and predict future sports talent using a **Random Forest Classifier**. The application provides data-driven insights to assist coaches, selectors, and recruiters in making informed talent scouting decisions.

---

## Features

- Cricket Player Performance Analysis
- AI-Based Talent Prediction
- Random Forest Machine Learning Model
- Interactive Prediction Dashboard using Streamlit
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis (EDA)
- Model Performance Evaluation
- Real-Time Player Talent Prediction

##  Model Used

**Random Forest Classifier**

Random Forest was selected because it:

- Handles large datasets efficiently
- Reduces overfitting
- Provides high prediction accuracy
- Performs well with structured sports performance data

## Installation

### Clone Repository

```bash
git clone https://github.com/Rohan Chauthe/AI_Sports_Talent_Scout.git
```

### Navigate to Project

```bash
cd AI_Sports_Talent_Scout
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
